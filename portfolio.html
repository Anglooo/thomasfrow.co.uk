Pressure Point Website | Code

A freelance web development job for a small massage and physiotherapist business in West Lothian. The site is created using the Bootstrap framework and makes use of some of WordPress' features.

Spectre

A software system created for my 3rd Year Group Project at Heriot Watt University. This project came runner up in the year, as judged by Scott Logic. The software is an oil-reservoir visualisation system which incorporates a web-front end and a Java back-end. I designed and implemented an algorithm that optimised data transfer between back-end and front-end on large sets of data.
